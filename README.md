# SakuraGakuin

All about a limited growth period only Japanese idol Sakura Gakuin.

## Installation

Add this line to your application's Gemfile:

    gem 'sakura_gakuin'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sakura_gakuin
